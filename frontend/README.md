# Frontend

Frontend da plataforma de inspiração visual, criado com React, TypeScript e Vite.

A página inicial apresenta os principais endpoints da API. As telas do produto ainda serão construídas a partir desta base.

```bash
npm ci
npm run dev
```

O servidor de desenvolvimento fica disponível em `http://localhost:5173`.

O endereço do backend é configurado em `.env`:

```env
VITE_API_URL=http://localhost:3000
```
